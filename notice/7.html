NGuard Security 2.0 오픈 베타 안내 NGuard Security 2.0 Open Beta Announcement
2024.01.01 00:00 (KST)

안녕하세요, NGuard 입니다.

저희 NGuard는 2024년 새해를 맞아 1년 반 동안 준비한 NGuard Security 2.0을 오픈 베타로 여러분들에게 선보이게 되었습니다.

NGuard Security 2.0은 더욱 사용성 향상에 초점을 두고 작업하였으며, 앞으로 더 많은 여러가지 기능이 추가될 예정입니다.

아직 모든 기능을 이용하실 수는 없지만, 최대한 빠르게 모든 기능을 이용하실 수 있도록 저희 팀원 전체는 노력하겠습니다.

현재 새로워진 콘솔(구 대시보드)와 보안 초대링크는 90% 이상의 기능을 사용하실 수 있습니다.
(보안 초대링크의 “본인인증” 기능 추가와 ”커스텀 도메인“ 기능 외 모두 사용 가능)

디스코드 봇은 최대한 빠르게 마저 작업하여 2월까지 리뉴얼하여 선보일 수 있도록 하겠습니다.

여러분의 많은 관심 부탁드리며, 아직 베타인 만큼 버그가 많습니다.
버그와 오류를 발견하시면 지체하지 말고 채널톡 등 문의 창구를 통해 제보 부탁드립니다.

새해 복 많이 받으세요.
NGuard 팀 올림

Dear customers, this is NGuard.

We're excited to bring you NGuard Security 2.0 in open beta, which has been a year and a half in the making.

We've been working on NGuard Security 2.0 with a strong focus on usability, and there are many more features to come.

Not all features are available yet, but our entire team is working hard to get them to you as soon as possible.

Currently, the new console (formerly known as the dashboard) and secure invite links are over 90% functional.
(Except for the addition of the "toss/SMS verify" feature in the secure invite link and the "custom domain" feature)

We'll be finishing up the renewal of the Discord bot as quickly as we can and hope to have it up and running by February.

We hope you enjoy it, but it's still as buggy as a beta.
If you find any bugs and/or errors, please don't hesitate to report them via the channel.io chat at the bottom of the homepage.

Thank you and Happy New Year.
